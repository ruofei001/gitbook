# 镜像切换

```
npm config set registry=http://registry.npmjs.org
npm config set registry=https://registry.npm.taobao.org
```

# 登录

```
npm adduser
Username:dwtt
Password:wangtao_……
Email:diwtt@126.com
```

# 发布

```
npm publish
```

# 删除包（发布时间太长了就不允许删除，可以看官网）

```
npm unpublish diwangtaotao  --force
```

# 删除包的某个版本

```
npm unpublish diwangtaotao@1.0.1  --force
```
